# KE-FORGE LEARN

AI-powered adaptive learning platform. Upload a PDF or paste a topic  
KE-FORGE LEARN splits it page-by-page, builds a personalized daily study 
plan, and adapts as you learn with quizzes, embedded videos, and an 
"I'm lost" button for instant ELI5 explanations.
Analytics lets you know your learning progress 

🔗 **Live demo:** https://ke-forge-learn.lovable.app

## Features
- 📄 Page-by-page PDF breakdown
- 📅 Adaptive daily intensity (light start, focused finish)
- 🎥 Embedded video + quiz reinforcement
- 🆘 "I'm lost" mode for instant simplified explanations
